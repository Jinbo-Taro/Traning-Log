# 屈中Ｐ重ねについて

何回も繰り返しているかもしれないが、最近の研究によると技の被せとしては有効だ。
相手のリバーサルを誘って失敗させたり投げ返しをミスらせたり。
但し無敵技をリバーサルされた場合は良くない。戻りフレームが０じゃないから。（0F じゃないから駄目という理屈も理解に苦しいけどな！）

戻りモーションを殴られていては何のために打撃重ねしているかという話で、昇竜持ちに対しては基本通りローキック重ねや地獄突き重ねをするべき。
Ｎキャラや無敵技がない相手には屈中Ｐ重ねが使える。屈中後に弱技に繋げれば投げハメの難度が下がる。

　  
　  

# 屈中Ｐ重ねた後の話

屈中Ｐを重ねた後は地獄突きからのボタン離しスクリューに移りたいところ。
しかし体の薄い一部キャラクターは地獄突きが届かない場合がある。
こういう時は遠立弱Ｐにすれば良いのか？ 遠立弱Ｐが屈状態の相手にヒットしたかどうかは確認が必要。

遠立弱Ｋに繋ぐのは微妙。近立弱Ｋのニーアタックが出てしまう可能性がある。

# ベガのニープレス対応

遠い間合いから出してくるタイミングが量れるのであれば、垂直ジャンプで躱すことができる。（下りでジャンプ攻撃が確定）
但し無謀な空ジャンプは下りをニープレスで狙われる。当たるか潰すかは状況次第でよくない。
また的確なタイミングでサイコクラッシャーを打たれると対応のしようがない。
あくまでニープレス一点読みで有効なハッタリ立ち回りだ。

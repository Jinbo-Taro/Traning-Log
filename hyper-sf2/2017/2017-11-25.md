# ダルシム戦あれこれ

基本

* 遠距離中Ｋ、強Ｋは屈中Ｐ、屈強Ｐなどで潰す
* 遠距離強Ｐ、遠距離中Ｐはダブラリで回避可能。屈中Ｐに潰される
* 遠距離強Ｐ、屈中Ｐは大足で潰す
* 遠距離立中Ｐは屈強Ｐで潰す


注意

* ダブラリは遠距離立強Ｐを回避できるが、先端を躱す位置だと遠立強Ｋに頭を蹴られる。偶に殴り返すことが出来るけど完全に運
* ヨガファイヤーをダブラリで回避すると頭を蹴られる


課題

* 屈強Ｐは全体フレームが大きい。スカりを待たれると確定で殴られるため、当てずっぽうで振るのは駄目


考察

* 屈中Ｐは隙が小さめだが当たりが弱い（気がする）。相打ちしても嬉しくない
* 屈弱Ｐは潰せる技と潰せない技に分かれる。相手の技を絞れるという筋が作れるか？
* ギリギリの間合いで立ち回っているとズームパンチの先端が下段判定になる。屈ガードしようとすると食らい判定が大きくなって触ってしまう。遠立強Ｋが活路か


開発中メモ

* 遠距離立強Ｐがギリギリ届かない間合いでヨガファイヤーを打たれた場合、前に飛び込むとダルシムの硬化を殴れる
* 投げの匂いが濃厚になるとダルシムは全力で回避行動を取る。読み合いをせず様子を見ることで後出し対応が出来るかもしれない。
